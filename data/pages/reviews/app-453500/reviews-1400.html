\u3010Frio - Lost in old town\u3011Pass with Green light .A single independent game .
Foreword:

I was in the store to see this upcoming game, I saw the screenshot and feel a bit interested, so I bought the game, and then spent a few hours to play.

I think the quality of this horror game is normal,I will point out some of the problems in the game and personal experience for the author and other players reference.
----------------------------------------------------------------------------
Personal feelings:

The first in-game animation tells why the girl enters the old town.

Girl's module is not bad,And girls voiceover too.
Also can change the clothing after the clearance.
This became my motivation to play this game.

Later I found that this game actually has support for multiple languages,
Unfortunately, after playing a bit found quite a few problems,
So I still set back to English.

The game has a lot of puzzle,
Need to pass through tools and basic math,
Basically a lot of tools will be reused.

Even the game is hard mod,
After you understand the mechanism of the ghost is actually very simple to dodge.

But after the game clear i still not very understand the story.
And there are many places in the game such as translation problems need to be modified.

Look forward to the author can do better day by day.
----------------------------------------------------------------------------
Game problem:

1.Not very scare
The horror atmosphere of the game,
Small space gives the player a repressed feeling,
Plus ghost will appear from time to time,
Sometimes it can scare people,but nothing more
When players are familiar with the routines of the ghosts,
It is not easy to be scared again,
This is like FNaF.

On this point, many horror games have the same practice so I can not say anything,
But the main problem is that players will not be scared by the ghost.

2.No obvious story
The game is short,The entire map is not large,
Are mainly based on the puzzle,
And it's too hard to understand what the game's story is saying.

I often play horror JRPG (ex: blue demon \u9752\u9b3c),
Even though all are 2D pixel monsters,
Terror level is not bad than 3D monster.
A good story can made a good horrible monster,
Just like the first question.player will not scare to the ghost.

3.Module dislocation
Sometimes characters,
Will move to the wrong place,
However, the game will automatically return to the correct place,
Although it does not affect the game,
Of course it is better not to have this problem.

4.Translation problem
Basically Chinese translation will result game puzzles can't be pass.
Don't know other languages have the same problem.

5.Button problem (Fix already)
the button looks like a mobile version, too large for the computer version.
----------------------------------------------------------------------------
postscript:
I heard that the author has already started planning the second episode,
Then first wish him well to complete the game,
A person can do this is already very good,
Of course, it can be better.\t\t\t\t\t
