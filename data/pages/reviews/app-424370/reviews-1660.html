\r\n\t\t\t\t\tIt's great so far, the good:
+ combat animation is one of the best in this genre, especially melee.
+ gameplay is addictive, generally trash mob lives longer so it will be dangerous if you are focused or surrounded. Elite/Boss has some huge damage/wind up attack, forcing you walk around not just attack spam.
+ environment is beautiful, much better than alpha
+ overall character growing feels fun.

things that could be better:
- lag. need servers for different regions (in Asia here, the latency is unbearable in peak hour). since this game is heavy action oriented, latency just makes you impossible to beat hard bosses.
- current skill design are generic, and skill upgrades are boring too.
- if skills have to locked strictly behind specific weapon set, you need A LOT more skills to make the game fun and replayable.
- range play is brain-dead easy (not sure about end game)\t\t\t\t\t
